# CRDIT Economy

## CRDITとは
CRDIT（クレジット）とは、広大なエコシステムを形成するための高度なガバナンス機能が実装された汎用トークンです。ERC20の仕様に基づいて実装されています。

## CRDITの基本情報
CRDITの基本情報は、以下の通りです。
- ティッカー：CRDIT
- 名称：Credit v1（クレジット v1）
- ブロックチェーン：BNBチェーン
- 発行上限：100,000,000 CRDIT

## CRDITの特徴
CRDITには、以下のようなユニークな特徴があります。
- ブラックリスト
- 税金
- 発行体
